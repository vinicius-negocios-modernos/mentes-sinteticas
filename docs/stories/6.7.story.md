# Story 6.7: Voice Mode — TTS/STT

**Epic:** 1 -- Transformacao Mentes Sinteticas
**Fase:** 6 -- Features Legendarias
**Status:** Draft
**Estimativa:** 10 SP (1 SP = ~2h)
**Criado por:** @sm (River)

## Descricao

Como usuario do Mentes Sinteticas, quero interagir por voz com as mentes usando microfone (STT) e ouvir respostas em audio (TTS), para que a experiencia de conversa seja mais natural, imersiva e acessivel.

Voice Mode adiciona capacidades de Text-to-Speech (TTS) e Speech-to-Text (STT) ao chat, permitindo que o usuario dite mensagens por voz e ouca as respostas das mentes em audio. Cada mente possui configuracao de voz distinta (pitch, rate), reforçando sua personalidade unica. V1 utiliza exclusivamente a Web Speech API (SpeechRecognition + SpeechSynthesis), sem custos de cloud, com fallback graceful quando as APIs nao estao disponiveis.

## Contexto

- O chat ja possui streaming via Vercel AI SDK v6 (`src/app/api/chat/route.ts`)
- Componentes de chat existentes em `src/components/chat/` (chat-interface, chat-message, chat-input, chat-header)
- Sistema de haptics (`src/lib/haptics.ts`) ja respeita `prefers-reduced-motion`
- A11y consolidada: ARIA live regions, focus-visible, semantic landmarks
- i18n preparado com `t()` e mensagens pt-BR (`src/lib/i18n/`)
- Mobile-first: design responsivo com touch targets de 44px minimo
- 6 mentes com personalidades distintas — cada uma tera voz configurada
- Web Speech API: `SpeechRecognition` (STT) disponivel em Chrome, Edge, Safari 14.1+; `SpeechSynthesis` (TTS) disponivel em todos os browsers modernos

## Notas Tecnicas

### Web Speech API — Suporte

| API | Chrome | Edge | Firefox | Safari |
|-----|--------|------|---------|--------|
| SpeechRecognition | Yes (webkit prefix) | Yes | No | Yes (14.1+) |
| SpeechSynthesis | Yes | Yes | Yes | Yes |

- STT requer `webkitSpeechRecognition` em Chrome/Edge (prefixed)
- STT em Firefox: botao de mic nao renderiza (graceful degradation)
- TTS disponivel em todos os browsers modernos
- Mobile: permissao de microfone requer HTTPS (ja em producao via Vercel)

### Arquitetura

```
src/lib/voice/

  speech-recognition.ts # STT wrapper (SpeechRecognition)
  speech-synthesis.ts   # TTS wrapper (SpeechSynthesis)

src/components/chat/
  chat-input.tsx         # + Mic button
  chat-message.tsx       # + Speaker button
  chat-header.tsx        # + Voice Mode toggle
```

### Decisions

- **V1: Web Speech API only** — zero custo, funciona offline (TTS), sem dependencias externas
- **V2 (futuro):** Cloud TTS (Google Cloud TTS ou ElevenLabs) para vozes de alta qualidade
- **Sem persistencia de preferencia na V1** — Voice Mode e per-session only
- **Auto-play aguarda streaming completar** — nao lê chunks parciais (evita audio entrecortado)
- **Mic timeout: 30s** — evita gravacao infinita acidental
